Controls:
WSAD - move camera
Left mouse button - place room, apply action
Right mouse button - select room, cancel action
Esc - cancel action
M - move room
R - resize room
U - increase room height
J - decrease room height
F1 - toggle draw room links
Delete - delete room
Num+ - move grid up
Num- - move grid down
Ctrl+N - new level
Ctrl+O - open level
Ctrl+S - save level
Ctrl+Shift+S - save level as
Ctrl+U - unlock cursor
Alt+F4 - exit
