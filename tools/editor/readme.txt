Controls:
WSAD - move camera
Left mouse button - place room, apply action
Right mouse button - select room, cancel action
Esc - cancel action
M - move room
R - resize room
Delete - delete room
Ctrl+N - new level
Ctrl+O - open level
Ctrl+S - save level
Ctrl+Shift+S - save level as
Ctrl+U - unlock cursor
Alt+F4 - exit
